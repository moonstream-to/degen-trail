# DegenTrailStats

This struct represents the stats of any NFT (wagons, items, etc.) that is used in The Degen Trail.


```solidity
struct DegenTrailStats {
    uint256 kind;
    uint256 speed;
    uint256 fight;
    uint256 repair;
    uint256 recovery;
}
```
